Project Setup

In order to start any of the projects on this site, you'll first need to create an empty project. The process can be kind of confusing, so refer to this page anytime you need to start a new project and you'll be fine.

Folders

The first thing you want to do is create your project folder structure. It's actually not that complicated, just pick a location on your computer where you want to store the project. Create a new folder and name it something descriptive. My folder here is called ProjectName. That's not very descriptive so choose a better name depending on what project you're working on.


Go into the folder you just created and create 2 more folders called 'include' and 'source'. The source folder will hold all of our C++ (.cpp) files and the include folder will hold all of our header (.h) files.


You also want to include the Makefile which can be found in your devkitpro folder structure at devkitpro/examples/gba/template folder. Just copy that Makefile into your project folder.

If, for some reason, you don't see the Makefile in your devkitpro, then you can just download it directly from github. The link below will take you there.

Go here to learn how.
