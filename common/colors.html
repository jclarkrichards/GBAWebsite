GBA Colors

The Gameboy Advance can display a good number of colors. Sure, it can't display millions of colors like your computer can, but it can at least display tens of thousands of colors. That's a lot more than its predecesor, the Gameboy Color.

So how many colors can the Gameboy Advance display? To answer that question we need to know how many bits are required to store a color into memory.

Take a look at the following bit layout. We can see that there are 16 bits here or 2 bytes. The Gameboy Advance uses 2 bytes to store color information. However, out of those 16 bits it really only uses 15 bits. That's why the last bit you see has an x, that bit is never used and is always 0. The reason it isn't used is because the Gameboy Advance stores colors in RGB format where each color channel uses 5 bits. So you use 5 bits for red, 5 bits for green, and 5 bits for blue. Which bits correspond to which color is shown in the diagram.

This means that each color channel can have 32 different values since 25 = 32. Also, since we only use 15 bits to represent a color, we can only have 215 = 32,768 colors represented on the Gameboy Advance. Like I said before, not a ton of colors, but good enough.
