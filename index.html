GBA Tutorials

Hopefully the domain is self explanatory, but this site is all about programming for the Gameboy Advance via small tutorials. If you're new here, then welcome! You should review the basics, then start with the projects in Mode3 first to get a handle on how things work.

Programming for the Gameboy Advance is unlike any other programming you've done most likely. For the most part it's just programming in C or C++, but with the added complexity of having to work directly with the memory on the GBA. So hopefully you have a strong handle on pointers. If you don't then check out the concepts page to learn what a pointer is and how to use it, especially in GBA land.

Now this website isn't meant to cover everything on the GBA, you likely won't find much history here. But you'll learn enough to complete and understand the simple projects that are presented here. If you're interested in learning more like the real nuts and bolts, then there are other websites that do that.

I started this site because I had dozens of GBA projects that I was working on and just wanted to share them. If you enjoy them then great, if not then that's fine too. I am not an authority on the GBA by any means and I'm not even the best programmer. I do my best with what I know though, besides I think programming for the GBA is fun so that's all the motivation I need.
